# DuctExact — HVAC 덕트 전개도(컷팅 도면) 생성기

형상을 선택하고 치수·심방식·단위를 입력하면 **DXF 컷팅 도면 + 치수표**를 생성하는
Windows 데스크톱 도구.

## 지원 형상
- 사각 직관 (Rectangular straight)
- 원형 직관 (Round straight)
- 사각 엘보 (Rectangular radius elbow) — 측판 2장 + 목/등 외피
- 원형 엘보 (Gored/새우관) — n조각 사인곡선 전개
- 리듀서/천이 — 원형→원형(콘), 사각↔원형(삼각분할, 변중앙 심 2조각), 사각→사각(4면)
- 원형 티/와이 — 분기관 새들 전개 + 본관 구멍 템플릿 (90°=티, 그 외=와이)
- 사각 티 — 분기관 전개 + 본관 직사각 구멍

## 주요 기능
- **단위 개별 선택**: 입력칸마다 mm/cm/m/in/ft, 출력 단위도 선택
- **ID/OD 보정**: 치수기준(내경/외경) + 게이지 두께로 중립선 보정,
  K-factor·굽힘여유 토글
- **네스팅**: 시트 크기 지정 → 패널 자동 배치(셸프 패킹, 90° 회전 옵션),
  활용률 표시, 네스팅 DXF 출력
- **완성도(조립도)**: 같은 치수로 정투상 완성도(정면/측면/단면) + 치수선 자동 기입.
  우측 "완성도" 탭에서 미리보기, 완성도 DXF 출력(실제 DIMENSION 엔티티)

## 설치 / 실행
```bash
pip install -r requirements.txt
python run.py
```

## 사용 흐름
1. 좌측에서 형상 선택
2. 가운데 치수 입력 (칸마다 단위 mm/cm/m/in/ft 개별 선택)
3. 공통 설정: 심방식 · 단부연결 · 게이지 · 출력단위 · 치수기준(ID/OD)
4. **전개도 생성** → 우측 미리보기 + 치수표
5. **DXF / CSV 내보내기**

DXF 레이어: `OUTLINE`(절단선) · `FOLD`(접기선) · `TEXT`(주석).

## 제작 여유값 (중요)
`ductexact/config/defaults.json` 의 심/단부/게이지 값은 SMACNA 통상 **추정치**입니다.
실제 절단 치수에 직접 반영되므로 **현장/공장 표준값으로 교체**하세요.
