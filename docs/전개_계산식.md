# DuctExact 전개 계산식 (검증용 레퍼런스)

각 형상의 전개도(컷팅 패턴) 계산식을 코드(`ductexact/core/shapes/`) 기준으로 정리.
외부 도구(엑셀/CAD/수계산)로 교차 검증할 수 있도록 변수 정의·단위·여유 처리를 명시한다.

> **단위 규약**
> - 모든 길이: **mm** (내부 표준). 입력 단위(mm/cm/m/in/ft)는 계산 전 mm로 변환.
> - 모든 각도 계산: **라디안**. `θ_rad = θ_deg · π / 180`.
> - 아래 식의 `W, H, D, Dm, Db ...` 는 모두 **중립선 보정 후** 값(§0).

---

## 0. 공통 — 중립선 보정 & 제작 여유

### 0.1 중립선 보정 `to_neutral(value)`
판재 두께 때문에 입력 단면치수(ID/OD)를 전개 기준선(중립선)으로 보정한다.

- `t` = 게이지별 판 두께(mm) — 게이지 두께표 조회
- `k` = 굽힘여유(중립선) 사용 시 K-factor, 미사용 시 `0`

| 치수 기준 | 보정식 |
|---|---|
| ID(내경) | `neutral = value + 2·k·t` |
| OD(외경) | `neutral = value − 2·(1−k)·t` |

> 기본 설정(굽힘여유 OFF, `k=0`): ID는 입력값 그대로, OD는 `value − 2t`.

### 0.2 여유값 (표에서 조회)
- `seam` : 심방식별 여유. 전개 **폭**에 **1회** 가산.
- `end` : 단부연결별 여유. 길이 방향 **양 끝에 각각** 가산.

| 심방식 | seam(mm) | | 단부연결 | end(mm) | | 게이지 | t(mm) |
|---|---|---|---|---|---|---|---|
| Pittsburgh | 22.0 | | Angle Flange + Bolt | 0.0 | | 18 | 1.31 |
| Snap Lock | 16.0 | | TDC | 9.5 | | 16 | 1.61 |
| Button Punch | 16.0 | | TDF | 9.5 | | 22 | 0.85 |
| Drive Cleat | 12.0 | | S&D (Slip & Drive) | 6.0 | | 20 | 1.00 |
| None | 0.0 | | None | 0.0 | | 24 | 0.70 |

> 위 수치는 SMACNA 통상 **추정치**(`config/defaults.json`). 현장 표준이 있으면 교체.

## 1. 사각 직관 (Rect straight)
입력: W(가로), H(세로), L(길이)

```
전개 폭  dev_w = 2·(W + H) + seam
전개 길이 dev_l = L + 2·end
면적     A     = dev_w · dev_l
접기선(길이방향) : x = W, W+H, 2W+H   (4면 경계)
                  x = 2(W+H)          (심 경계)
```

## 2. 원형 직관 (Round straight)
입력: D(지름), L(길이)

```
원주     circ  = π · D
전개 폭  dev_w = circ + seam
전개 길이 dev_l = L + 2·end
```

---

## 3. 사각 엘보 (Rect elbow / SREL)
입력: W(곡면방향 가로), H(외피 폭=세로), Rt(목반경), θ(각도), leg(단부 직관)

```
등반경    Rh        = Rt + W
목 호길이  throat_len = Rt · θ_rad
등 호길이  heel_len   = Rh · θ_rad
```

**HEEL / THROAT 외피 (각 1장, arc = 해당 호길이)**
```
전개 길이 = 2·end + 2·leg + arc
전개 폭   = H + seam
절곡선(길이방향): x = end, end+leg, end+leg+arc, (전개길이−end)
```

**측판 CHEEK (2장)**
```
동심 원호: 내측 반경 Rt, 외측 반경 Rh, 스윕각 θ_rad
양 끝 직관: 길이 = leg + end (호 끝에서 접선 방향으로 연장)
```

> 완성도(측면도)는 같은 Rt·Rh·θ·leg 실루엣 + W×H 단면도.

## 4. 원형 엘보 (Gored / 새우관)
입력: D(지름), R(중심선 반경), θ(각도), n(분할수)

```
r = D / 2
조인트당 꺾임 β = θ_rad / (n − 1)
컷각        α = β / 2
미터 진폭    a = r · tan(α)

끝조각 중심선 길이 = R·β / 2     (half_len = R·β/4, 2개)
중간조각 중심선 길이 = R·β        (half_len = R·β/2, n−2개)

전개 폭 = 2π·r + seam = π·D + seam
조인트 수 = n − 1
```

**미터 절단선(전개 곡선)** — 둘레각 `φ = 2π·i/n`, 가로 `x = (π·D)·i/n`
```
상단 y = +half_len + a·cos φ
하단 y = −half_len − a·cos φ
(끝조각의 평평한 연결단은 a=0; 그 단에 end 가산)
```

## 5. 리듀서 — 원형 → 원형 (편심 콘 전개 = 부채꼴)
입력: D1(입구경), D2(출구경), L(길이)

```
R1 = D1/2, R2 = D2/2
big = max(R1,R2),  small = min(R1,R2)
빗변      slant = √( L² + (big − small)² )
큰 호 반경  Rb   = slant · big / (big − small)
작은 호 반경 Rs   = Rb − slant
부채꼴 각도 φ    = 2π · big / Rb        (rad)
```
전개: 반경 `Rs ~ Rb`, 사잇각 `φ` 인 부채꼴(환형 섹터).

## 6. 리듀서 — 사각 ↔ 원형 (실길이 삼각분할, 2조각)
입력: 사각 W×H, 원형 D, 길이 L  (원형→사각은 입·출구만 반대, 전개 동일)

닫힌 공식이 아니라 **3D 실길이 삼각분할**. 검증 핵심값:
```
사각 4꼭지점 (z=0): (±W/2, ±H/2, 0)
원형 점 (z=L):      P(a) = (r·cos a, r·sin a, L),  r = D/2
각 변 접점:  M0=(0,−r,L), M1=(r,0,L), M2=(0,r,L), M3=(−r,0,L)
```
- 평면 삼각형 ×4 : 밑변 = 사각 한 변, 꼭지 = 해당 접점 M
  - 슬랜트(꼭지→변 끝) 실길이 `= √( (cx−Mx)² + (cy−My)² + L² )`
- 코너 콘 ×4 : 꼭지 = 사각 코너 C, 윗변 = 인접 접점 사이 1/4 원호
  - C→호점 실길이 `= √( (Cx − r·cos a)² + (Cy − r·sin a)² + L² )`

전개: 변 중앙(접점 방향)을 심선으로 좌/우 **2조각(각 ~180°)** 으로 실길이 삼각분할.
(동심 기준; 편심 미지원)

## 7. 리듀서 — 사각 → 사각 (4면)
입력: 입구 W1×H1, 출구 W2×H2, 길이 L

```
앞/뒤면 (W면, ×2): 사다리꼴  밑변 W1, 윗변 W2, 면높이 = √( L² + ((H1−H2)/2)² )
좌/우면 (H면, ×2): 사다리꼴  밑변 H1, 윗변 H2, 면높이 = √( L² + ((W1−W2)/2)² )
```

## 8. 원형 티 / 와이 (Round tee)
입력: Dm(본관경), Db(분기경), Lb(분기길이), α(분기각, 90°=티)

```
R = Dm/2, r = Db/2,  α_rad = 분기각
분기 둘레각 φ ∈ [0, 2π]
A = r·cos φ
B = r·sin φ · cos α

분기축 거리(본관 표면까지)
  s(φ) = ( −B + √(R² − A²) ) / sin α
```

**분기관 새들 전개**
```
폭        = π·Db + seam
바닥 곡선  y = s(φ) − s_min          (x = π·Db · i/n)
자유단(윗변) = Lb + max(s) − s_min + end
새들 최대 깊이 = max(s) − min(s)
  └ 90°(티): s(φ) = √(R² − r²·cos²φ),  깊이 = R − √(R² − r²)
```

**본관 구멍 템플릿** — 분기 벽면 3D 점 → 본관 표면 전개
```
x = s·cos α − r·sin φ·sin α
y = r·cos φ
z = s·sin α + r·sin φ·cos α
전개 좌표:  U = R · atan2(z, y),   V = x
```

## 9. 사각 티 (Rect tee, 90° 평면 본관)
입력: 본관 Wm×Hm, 분기 Wb×Hb, 분기길이 Lb

```
[분기관]  전개 폭 = 2·(Wb + Hb) + seam
          전개 길이 = Lb + end (자유단)
          접기선: x = Wb, Wb+Hb, 2Wb+Hb (4면), 2(Wb+Hb) (심)
[본관 구멍]  Wb × Hb 직사각형 (평면 본관 가정; 곡면 본관은 새들 보정 필요)
```

## 검증 시 유의사항
1. **각도는 라디안.** 호길이(`R·θ`), 부채꼴 각도 등 모든 곳에서 `θ_rad` 사용.
2. **W·H·D는 중립선 보정 후 값.** 기본(굽힘여유 OFF·ID 입력)은 보정이 0이라 입력값과 동일하지만, OD 입력이나 굽힘여유 ON이면 달라짐(§0.1).
3. **seam은 폭에 1회, end는 길이 양끝에 각각.** 표의 "심 여유/단부 여유" 행 값과 대조.
4. 리듀서 사각↔원형·사각→사각, 사각 티는 **근사/평면 가정**(편심·곡면 본관 미반영).
